# 贪吃蛇

一个经典的贪吃蛇游戏，使用 Python + Pygame 实现。

## 环境要求

- Python 3.8+
- Pygame

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python snake.py
```

## 操作

- 方向键 / WASD — 控制蛇的移动
- R — 重新开始
- ESC / Q — 退出

## 项目结构

```
贪吃蛇/
├── snake.py          # 主程序
├── game.py           # 游戏逻辑
├── config.py         # 配置常量
└── requirements.txt  # 依赖
```
